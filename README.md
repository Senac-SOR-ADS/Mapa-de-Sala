# Mapa de Sala
ferramenta criada para gerenciamento de reserva de mapa de sala no Senac Sorocaba

## Rodando localmente

#### Configure o ambiente virtual
```bash
  python -m venv env
```

#### Ative o ambiente virtual

###### PowerShell
```bash
  env\Scripts\Activate.ps1
```
###### CMD
```bash
  env\Scripts\Activate.bat
```
